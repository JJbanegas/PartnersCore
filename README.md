# PartnersCore

Microservice responsible for managing partner-related operations within the platform.

Handles the lifecycle of partner service requests, including creation, tracking, and coordination between partners and clients.

## Responsibilities
- Partner service request management
- Request lifecycle and status handling
- Coordination with other platform services

## Architecture
Follows the same layered architecture principles as the platform (Domain, Application, Infrastructure, Presentation).

## Status
Work in progress — MVP stage.
